# 🚀 Startup Funding Analysis Dashboard

An interactive **Streamlit** web application for analyzing startup funding trends, investors, and investment patterns using a cleaned startup funding dataset.

The dashboard enables users to explore funding statistics, investor portfolios, startup insights, and yearly investment trends through an intuitive interface and visualizations.

---

## 📌 Features

### 📊 Overall Analysis
- Total funding raised
- Highest funding received by a startup
- Average funding per startup
- Total funded startups

### 🏢 Startup Analysis
- Browse startups from the dataset
- Foundation for displaying startup-specific insights

### 💰 Investor Analysis
- View recent investments
- Biggest investments by startup
- Sector-wise investment distribution
- Year-over-Year (YoY) investment trend

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/Startup-Funding-Analysis.git
```

Move into the project directory

```bash
cd Startup-Funding-Analysis
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
